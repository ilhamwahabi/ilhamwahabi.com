# ilhamwahabi.com

Place where I will share my thoughts, talks, and projects.

Developed using:

- Remix
- TailwindCSS
- Notion
- Vercel
- Vercel Analytics.

## Development

```sh
npm install
```

```sh
npm run dev
```

Open up [http://localhost:3000](http://localhost:3000) and you should be ready to go!
